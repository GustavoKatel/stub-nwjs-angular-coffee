stub-nwjs-angular-coffee

Package "ready-to-go" to develop applications in NW.js, Angular.JS and CoffeeScript.

Features

  • Auto download NW.js builds using grunt-node-webkit-builder
  • Create distribution packages
  • Development and Production environments
  • Dependencies management with bower
  • Compile with grunt
  • AngularJS
  • CoffeeScript
  • NW.JS

How to

  1. Install dependencies: bower install and npm install
  2. Build grunt build[:env].
  3. Create dist packages grunt dist[:env].
  4. Run grunt app[:env].

Possible values to env are dev and prod
